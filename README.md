# 便捷设置插件 (easystart)

## 产品简介

OpenWrt 便捷设置插件，支持多种常用路由模式的一键切换：
- 有线接入：路由模式、旁路由模式
- 无线接入：中继模式、桥接模式

## 核心功能

- **零代码配置**：普通用户无需手动修改网络、DHCP、防火墙等复杂配置
- **一键切换**：单页面选择模式，10 秒内完成配置
- **自动适配**：自动识别网络环境，无需手动填写复杂参数
- **配置安全**：切换前自动备份原有配置，支持一键回滚
- **轻量化**：不占用系统资源，无后台常驻
- **可视化反馈**：实时显示当前工作模式、网络状态
- **WiFi 设置**：支持 2.4G/5G 合一，快速配置 WiFi 名称和密码
- **无线扫描**：支持 WiFi 扫描和连接测试功能
- **设备支持检查**：自动检测设备是否支持无线功能，禁用不支持的选项

## 环境要求

- 系统：OpenWrt 21.02 及以上稳定版
- 架构：x86_64、ARM、MTK 等通用架构

## 安装方法

### 方法一：通过 OpenWrt 软件库安装

```bash
opkg update
opkg install luci-app-easystart
```

### 方法二：手动安装

1. 下载编译好的 IPK 包
2. 上传到 OpenWrt 路由器
3. 执行安装命令：

```bash
opkg install luci-app-easystart_1.1-0_all.ipk
```

## 使用方法

1. 登录 OpenWrt 后台管理界面
2. 进入 **网络 → 便捷设置**
3. 在 **联网设置** 页面选择需要的工作模式：
   - 有线接入：路由模式、旁路由模式
   - 无线接入：中继模式、桥接模式
4. 填写必要的参数（如 PPPoE 账号、旁路由静态 IP、WiFi 名称和密码）
5. 点击 **一键应用配置** 按钮
6. 等待网络服务重启（约 3-5 秒）
7. 配置生效，路由器将按照选择的模式工作

## 模式说明

### 1. 有线接入

#### 路由模式（主路由）
- **适用场景**：路由器直接连接光猫，作为家庭/办公核心主路由
- **自动配置**：
  - 启用 WAN 口，支持 PPPoE 拨号/动态 IP/静态 IP 三种上网方式
  - LAN 口开启 DHCP 服务器，自动分配内网 IP
  - 开启 NAT 转发、防火墙默认规则
  - 网关/DNS 指向路由器自身

#### 旁路由模式
- **适用场景**：配合主路由使用，仅负责科学上网、广告过滤、内网管理等增强功能
- **自动配置**：
  - 禁用 WAN 口，仅保留 LAN 口
  - LAN 口设置静态 IP（与主路由同网段）
  - 开启 DHCP 服务器，网关/DNS 强制指向本旁路由
  - 关闭冗余防火墙规则，开启 IP 流量转发
  - 提示：请关闭主路由 DHCP（避免冲突）

### 2. 无线接入

#### 中继模式
- **适用场景**：路由器通过无线方式连接上级路由器，扩展网络覆盖范围
- **自动配置**：
  - 扫描并连接上级 WiFi
  - 配置本地网络参数
  - 开启 DHCP 服务器
  - 实现无线信号中继

#### 桥接模式（无线 AP）
- **适用场景**：路由器仅做无线扩展，无路由功能
- **自动配置**：
  - WAN 口与 LAN 口桥接（合并为一个局域网）
  - 完全关闭 DHCP 服务器
  - 网关/DNS 指向上级主路由
  - 关闭防火墙、NAT 功能
  - 设备变为纯二层 AP，由主路由统一分配 IP

## 技术实现

- 基于 OpenWrt 原生 UCI 配置系统
- 采用 Shell 脚本自动化执行配置
- 兼容所有 OpenWrt 架构（x86/ARM/MTK）
- 无依赖、无第三方库，原生 OpenWrt 直接运行

## 常见问题

### Q: 切换模式后网络断开怎么办？
A: 请手动登录路由器后台，重新配置网络设置。

### Q: 旁路由模式下无法上网怎么办？
A: 请确保：
1. 旁路由 IP 与主路由在同一网段
2. 已关闭主路由的 DHCP 服务器
3. 设备已获取到旁路由分配的 IP 地址

### Q: 桥接模式下无法获取 IP 怎么办？
A: 请确保上级主路由的 DHCP 服务器已开启，且设备与路由器的物理连接正常。

### Q: 无线接入模式下扫描按钮不可用怎么办？
A: 这表示您的设备不支持无线功能，请检查设备是否配备无线网卡。
